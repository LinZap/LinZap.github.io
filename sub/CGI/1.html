簡介

CGI environment variables是CGI程式(某些CGI environment variables在其他server端技術亦是存在的)執行時所處的環境資訊，是程式執行時重要的輸入資料，CGI environment variables共可分成三種類別

user

此類環境變數記錄有關browser之資訊，例如HTTP_USER_AGENT記錄使用者使用什麼瀏覽器

server

此類環境變數記錄有關Web server之資訊，例如SERVER_SOFTWARE記錄伺服器使用的軟體

request

此類環境變數記錄有關請求之資訊，例如QUERY_STRING記錄要送給Server之查詢字串


撰寫一隻取得 CGI 環境變數的程式

使用Perl

#!/usr/local/bin/perl

print "Content-type:text/html \n\n";

foreach $var (keys(%ENV)){

print "環境變數 $var內容是$ENV{$var} <p> \n"

}

print "你的 IP ADDRESS 是 $ENV{'REMOTE_ADDR'} <p> \n";
