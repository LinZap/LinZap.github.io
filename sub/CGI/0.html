簡介

The Common Gateway Interface (CGI) is a standard for interfacing external applications with Web servers

The external applications are typically called CGI scripts or CGI programs

From the browsers' side, users percept that CGI programs (or other server side programs) are extension of Web Servers.

Typically, when a CGI program is requested, the Web server creates a process to run the CGI program. (Some other approaches like FastCGI are proposed to improve the performance)
