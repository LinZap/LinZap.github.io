自訂 JS Function

首先必須使用關鍵字 function 接著為其命名，並撰寫其要做的事情，往後便可重複呼叫該 function 以利程式撰寫

myfunc("hi~~~~");

function myfunc(arg){

alert(arg);

}

完整程式碼

}

</script>


</head>
