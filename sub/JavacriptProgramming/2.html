JS基本的物件

navigator

has properties for the name and version of the Browser being used, for the MIME types supported by the client, and for the plug-ins installed on the client

window

the top-level object; has properties that apply to the entire window. There is also a window object for each "child window" in a frames document

document

contains properties based on the content of the document, such as title, background color, links, and forms

location

has properties based on the current URL

history

contains properties representing URLs the client has previously requested
