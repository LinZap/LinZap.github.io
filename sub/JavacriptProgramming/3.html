Document Object Model

DOM (Document Object Model)
DOM為HTML及XML文件之標準文件模型，為了HTML及XML文件處理之需，W3C制定DOM(Document Object Model)之規格標準，規範一個HTML或XML文件為一個DOM tree，HTML或XML文件中每個元素(標籤)，每個元素之屬性及文字內容均是此DOM tree之node

DOM標準制定一套標準之物件集合體，規範一個HTML或XML文件中各物件(標籤)含那些標準的屬性(或物件)，含那些標準的methods，有那些標準的事件(event)可被觸發，這些規範形成一套標準API(Application Programming Interface)，一個程式可透過DOM API 讀取或更改一個HTML或XML文件中之元素(標籤)，元素屬性之內容等

DOM規格標準是跨平台跨語言，目前能支援DOM之程式語言已有許多，應用於client端之Javascript是最常見支援DOM之程式語言，Javascript程式可透過DOM API控制及處理一個HTML文件。有了DOM標準規格，處理HTML或XML文件之程式，才能具跨平台之優點。
