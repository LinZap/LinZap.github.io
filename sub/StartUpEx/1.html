簡介

當網頁需要藉由使用者透過某些操作後，改變網頁原先設定的樣式時，我們可以利用撰寫 JavaScript 程式來達到效果，下面將展現幾個基本案例。

改變文字樣式

點擊按鈕後，改變文字呈現的樣式

Demo：

Hello World！！

原始碼：

<script>
function dy_change_font(){
var df = document.getElementById('dyfont').style;
df.fontFamily = '微軟正黑體';
df.fontWeight = 'bolder';
df.color ='red';
}
</script>

<span id="dyfont">Hello World！！</span>
<button onclick="dy_change_font()">Click</button>


利用 CSS3 產生補間動畫

除了一般的改變元素外觀外，配合 CSS3 Transitions，更可呈現補間動畫的效果

首先，在畫面上建立一個 div 區塊，設定其基本外觀與 transition 並指定作用於 transform 改變時，
瀏覽器會花費 1 秒做出補間動畫效果。

再來，撰寫點擊後欲執行的功能。我們利用 .style 屬性來改變該物件外觀，transform 的 rotateY 可以進行水平翻轉。

最後，為了確保主流瀏覽器皆能正確執行，我們需要額外加上 -webkit- 的 CSS 宣告，包含Javascript的部分亦然。

Demo：

點我翻轉

原始碼：

<style type="text/css">
#dy_tion{
width : 65px;
height : 70px;
font-size : 28px;
background-color : #eae;
transition : transform 1s;
-webkit-transition : -webkit-transform 1s;
}
</style>

<script>
function dy_change_css(){
var tion = document.getElementById('dy_tion').style;
tion.webkitTransform = 'rotateY(-135deg)';
tion.transform = 'rotateY(-135deg)';
}
</script>

<div id="dy_tion" onclick="dy_change_css()">點我翻轉</div>
