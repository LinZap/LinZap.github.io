簡介

當網頁需藉由使用者某些操作後，才顯示出對應的資料時，便需要動態地改變網頁內容。
而這樣的功能通常藉由撰寫JavaScript程式來達到效果。，下面將展現幾個基本案例。

改變文字內容

點擊按鈕後，改變文字顯示內容

Demo：

還沒點！

原始碼：

<script>
function dy_change_cxt(){
// innerHTML 可以插入指定內容
document.getElementById('dycxt').innerHTML="已經點了！";
}
</script>

<span id="dycxt"> 還沒點！</span>
<button onclick="dy_change_cxt()"> Click </button>


兩數相加

輸入兩個整數，後方印出答案
首先，先在畫面上新増 2 個 input 元素，並將其註冊 onchange 傾聽事件
接下來，撰寫事件觸發後相加的功能，利用 .value 取得 input 元素的數值
最後，使用 .innerHTML 將結果印到畫面中。

Demo：

+ = 0

原始碼：

<script>
function sum(){
// parseInt可以將字串轉為整數
document.getElementById('dy_ans').innerHTML =
parseInt(document.getElementById('dy_num1').value) +
parseInt(document.getElementById('dy_num2').value);
}
</script>

<input type="number" value="0" id="dy_num1" onchange="sum()"> +
<input type="number" value="0" id="dy_num2" onchange="sum()"> =
<span id="dy_ans">0</span>
