Client端技術

Browser附加元件

內嵌在瀏覽器中用來延伸瀏覽器功能(如展現特殊媒體, 便捷的操作列)的程式模組 如：Flash Player, Java Plug-in, Google search toolbar

Browser Helper application

是一個完整的程式(complete program)，由瀏覽器呼叫來輔助展現網頁上的內容如：Word, Acrobat...

JavaScript

JavaScript為一個programming language，當用於client端，設計者可利用Javascript語言設計一些程式以控制或處理HTML文件內容

Java Applet

Java寫出來的程式可分為兩類，分別是Java Applet與一般的Application，而Application 這一類就與一般的程式如C++是類似的，就是一個獨立可執行的應用程式，而Java Applets雖也是一個程式，但它只能在瀏覽器中(或利用Applet Viewer)執行。網頁設計者可在網頁中夾帶一個Applet程式(利用＜Applet＞(標籤)，當此網頁被Browser下載時，該Applet程式也會被下載到瀏覽器中執行

ActiveX

ActiveX controls為MS Window平台之技術，一個ActiveX controls為一個Windows系統中之程式元件，在IE環境中，若一個網頁中內嵌有一個ActiveX control程式元件(利用＜Object＞(標籤)，當此網頁被下載後，此ActiveX control亦會被下載至瀏覽器端之電腦，系統驗證其來源與安全性，使用者同意後就可執行。 在Web系統中，ActiveX controls的目的基本上與Java Applet是類似的

Server端技術

PHP

PHP(Hypertext Preprocessor)是一種開源的通用電腦手稿語言，尤其適用於網路開發並可嵌入HTML中使用。PHP的語法借鑒吸收了C語言、Java和Perl等流行電腦語言的特點，易於一般程式設計師學習。PHP的主要標的是允許網路開發人員快速編寫動態頁面，但PHP也被用於其他很多領域

ASP

動態伺服器網頁(Active Server PagesASP)，由微軟公司開發的伺服器端執行的指令碼平台，它被Windows下Internet Information Services (IIS)的程式所管理

JSP

JSP(JavaServer Pages)是由Sun Microsystems公司倡導和許多公司參與共同建立的一種使軟體開發者可以響應客戶端請求，而動態生成HTML、XML或其他格式文件的Web網頁的技術標準。JSP技術是以Java語言作為手稿語言的，JSP網頁為整個伺服器端的Java庫單元提供了一個介面來服務於HTTP的應用程式

Perl

Perl是高階、通用、直譯式、動態的程式語言。最初設計者拉里·沃爾(Larry Wall)為了讓在UNIX上進行報表處理的工作變得更方便，決定開發一個通用的腳本語言
