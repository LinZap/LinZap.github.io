靜態網頁

所謂的靜態網頁(static Web contents) 是指一個網頁不論何時，何人去讀取均得到相同的內容
網頁作者設計該網頁時就已知網頁之內容，基本上一個名為xxx.htm(或.html)之網頁為一個靜態網頁，但此種講法會有些爭議

若xxx.htm網頁中含有Javascript code 則基本上可稱之為動態網頁，因其內容可能是由Javascript code 動態執行時產生的，因此，較關鍵之處在於在於該網頁內容是在請求時才經由程式產生(動態)，或是在網頁設計者設計時就已經確定內容(靜態)

在HTML格式的網頁上，有時會出現各種動態的效果，如：GIF格式的動畫、Flash、跑馬燈等，這些內容雖有動態效果，但只是視覺上的動態，應歸類為靜態內容較恰當。
